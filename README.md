## 👋 Who Am I?

I'm **Masudur Rahman Sifat**, a self-taught Python developer and automation engineer from **Bangladesh 🇧🇩**. I operate under the brand **Mr-SxR**, which stands for **Speciality & Reliability** — the two things I hold as the standard for every tool I build.

My journey started with a deep curiosity about hacking and how things work under the hood. That curiosity led me into cybersecurity, then programming, and eventually into building real software that solves real problems.

I'm entirely self-taught. No courses, no classrooms — just open-source code, endless experimentation, and a lot of persistence.
